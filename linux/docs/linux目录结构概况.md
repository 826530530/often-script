## linux 目录结构概况

### 一、linux “/”为linux根目录，该目录下主要文件夹为：
```bash
[root@localhost custom-user]# cd /
[root@localhost /]# ls
bin   dev  home  lib64  mnt  proc  run   srv  tmp  var
boot  etc  lib   media  opt  root  sbin  sys  usr
```
